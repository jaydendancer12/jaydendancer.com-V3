About Me

I’m a Software Engineer who enjoys turning ideas into real, working products! I started out building for the web, and over time that grew into a broader skill set that includes both web and mobile development, and AI Engineering!

I focus on full stack development using React, TypeScript, and Node.js, and I also build native mobile apps with Swift for iOS and Kotlin or Java for Android. Alongside that, I work with AI powered features, integrating LLM APIs, building data driven workflows, and using AI tools to move faster and design smarter solutions. This mix allows me to classify myself in the category that I would call - The Swiss Army Knives - of engineers!

I care a lot about clean, efficient code and solid development practices. I’m comfortable working with tools like Docker, MongoDB, and modern GitHub workflows, and I build systems that both scale and maintain.
